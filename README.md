# Nvim

cehnhy's neovim config

## Install

> Clone Repository

```shell
git clone git@github.com:cehnhy/nvim.git ~/.config/nvim
```

> Install Packer

```shell
git clone --depth 1 https://github.com/wbthomason/packer.nvim ~/.local/share/nvim/site/pack/packer/start/packer.nvim
```

> Install Plugins

```txt
:PackerSync
```
